# 🎯 FracShooter

**FracShooter** es un videojuego educativo desarrollado en Python con Pygame donde el jugador debe disparar a la fracción correcta según una operación dada.

Combina aprendizaje y diversión para reforzar el manejo de fracciones mediante un sistema de niveles, puntajes y retroalimentación inmediata.

## 🕹️ Jugar ahora

👉 https://richardcubas.github.io/FracShooter/

## 🚀 Características

* Operaciones con fracciones ( +, −, ×, ÷ )
* Niveles de dificultad
* Sistema de puntajes y récord
* Guardado y exportación de resultados
* Compatible con web (Pygbag)
